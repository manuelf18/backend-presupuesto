# backend-presupuesto
Backend para el proyecto de Calidad de Software, hecho en express.

## Run this project
1. Install dependencies (nodemon is optional but recommended)
  ``` 
  npm install
  npm install -g nodemon
  ```
2. Run
  ```
  nodemon
  ```
  or
  ```
  npm run test
  ```
  (both just run nodemon)
